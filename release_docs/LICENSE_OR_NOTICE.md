This desktop build is a project/demo release of Posture Detection App.

The app includes or depends on third-party Python packages such as OpenCV,
MediaPipe, TensorFlow, scikit-learn, CustomTkinter, Pillow, matplotlib, pandas,
NumPy, joblib, and statsmodels. Review each package license before public or
commercial distribution.

The bundled model and scaler are project artifacts:

- `models/ann_best.keras`
- `models/scaler.pkl`

The app is intended as an assistive posture reminder and research/demo product.
It is not a medical diagnosis tool.
